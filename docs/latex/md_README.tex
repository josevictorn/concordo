Esse projeto é um sistema com recursos similares ao Discord, porém vai funcionar somente em modo texto e sem recursos de rede.

O sistema é operado através de comandos de uma única linha, compostos por um nome, seguido de parâmetros.\hypertarget{md_README_autotoc_md1}{}\doxysection{Funcionalidades}\label{md_README_autotoc_md1}

\begin{DoxyItemize}
\item criar um usuário com nome, e-\/mail e senha;
\item autenticação de usuários;
\item sair do sistema;
\item desconectar da aplicaçao;
\item criar servidores;
\item mudar descrição de servidores;
\item setar código de convite para um servidor;
\item listar servidores do sistema;
\item remover um servidor;
\item entrar em um servidor;
\item sair de um servidor;
\item listar membros de um servidor;
\end{DoxyItemize}\hypertarget{md_README_autotoc_md2}{}\doxysection{Tecnologias utilizadas}\label{md_README_autotoc_md2}
